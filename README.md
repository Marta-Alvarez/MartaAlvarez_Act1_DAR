
# Relación entre HTML, CSS y JavaScript: Fundamentos de uso

Este proyecto web, disponible en [GitHub](https://github.com/Marta-Alvarez/MartaAlvarez_Act1_DAR), se ha desarrollado utilizando los tres pilares fundamentales del desarrollo front-end: **HTML**, **CSS** y **JavaScript**. A continuación se describe el papel de cada uno y cómo se han aplicado en el proyecto.

---

## 1. HTML (HyperText Markup Language)

El HTML constituye la **estructura base** de la web, actuando como el esqueleto que define qué contenido aparece en pantalla. Permite organizar elementos como títulos, imágenes, enlaces, listas y botones.

En este proyecto se han utilizado dos páginas principales:

- `index.html`: la página principal, obligatoria en cualquier sitio web.
- `enlaces.html`: una página secundaria que complementa el contenido.

Ambas páginas incluyen:

- Títulos definidos con la etiqueta `<h1>`
- Listas estructuradas con `<ul>` y `<li>`
- Enlaces de navegación
- Botones interactivos

---

## 2. CSS (Cascading Style Sheets)

El CSS se encarga del **diseño visual** del sitio. Controla cómo se presenta el contenido: colores, márgenes, tipografía, alineación, y otros aspectos estéticos.

En este proyecto, el archivo `style.css` define los estilos globales y se enlaza al HTML mediante:

```html
<link rel="stylesheet" href="style.css">

Gracias a CSS, se ha logrado centrar elementos, aplicar colores personalizados y mejorar la legibilidad del contenido.
